# dwm

a st forked from [yaocccc](https://github.com/yaocccc/st)

## Installation

```
git clone https://github.com/planetarium1001/st.git
cd dwm
sudo make clean install
```
## Keys

|Key|Description|
|:--:|:--:|
|Alt + l|Zoom in|
|Alt + h|Zoom out|
|Alt + j|Move down|
|Alt + k|Move up |
|Alt + r|Reset font size|
